**实验二：智慧养老项目核心模块开发的实验任务书（约8学时）**

**一、实验目的**

1. 掌握人工智能技术在智慧养老场景中的应用方法

2. 掌握计算机视觉核心算法的原理与实现

3. 培养综合运用多种AI技术解决实际问题的能力

4. 掌握AI系统从数据采集到模型部署的完整开发流程

**二、实验环境**

操作系统: Ubuntu 16.04+ 或 Windows 10+

开发环境: Python 3.6+、Anaconda

核心库: OpenCV、TensorFlow、Keras、dlib、face\_recognition

硬件要求: 摄像头、支持CUDA的GPU（可选）

**三、实验内容与时间安排**

**1. 基础环境搭建与人脸识别系统**

**任务1.1：智慧养老环境搭建**

安装Ubuntu操作系统及Python开发环境

配置OpenCV、TensorFlow等计算机视觉库

创建独立的Python虚拟环境

**任务1.2：人脸采集与识别系统**

实现摄像头实时人脸检测功能

开发多姿态人脸图像采集程序

构建人脸识别模型并训练

实现陌生人检测与报警功能

**交付物:**

环境配置文档

人脸采集程序

人脸识别模型文件

陌生人检测演示视频

**2. 情感分析与行为监测**

**任务2.1：老人情感状态分析**

使用KNN算法实现基础情感分类

构建ANN神经网络优化情感分析

采用CNN卷积神经网络提升准确率

实现实时面部表情识别

**任务2.2：老人行为监测**

开发老人摔倒检测模型

实现禁止区域入侵监测

构建老人义工互动识别系统

**交付物:**

情感分析模型（准确率>90%）

摔倒检测程序

行为监测演示视频

模型性能分析报告

**3. 系统集成与优化**

**任务3.1：多摄像头监控系统**

实现房间、走廊、院子多场景监控

开发摄像头画面实时显示功能

实现视频自动录制与存储

**任务3.2：数据管理与系统集成**

设计事件数据库结构

实现监测事件自动入库

开发数据查询与分析接口

**交付物:**

多摄像头监控系统

数据库设计文档

系统集成演示

事件记录查询界面

**4. 性能优化与项目展示**

**任务4.1：模型性能优化**

调整超参数提升模型准确率

优化算法运行效率

解决实际部署中的技术问题

**任务4.2：项目总结与展示**

整理完整项目文档

制作项目演示视频

撰写实验总结报告

**交付物:**

优化后的完整系统

项目演示视频

详细实验报告

技术问题解决方案文档

**1. 对毕业要求7.2的支持：**

要求内容：“能够评价计算机应用领域复杂工程问题的实践活动对社会可持续发展和环境造成的影响。”

**支持方式：**

实验2以“智慧养老”为实际应用场景，涉及人脸识别、情感分析、摔倒检测等多个AI技术的综合应用，属于典型的复杂工程问题。

伦理责任：误识别、误报警可能带来的后果。

通过撰写实验报告中的“创新性与实用性”部分，学生可系统评估该技术的社会影响与可持续性。
